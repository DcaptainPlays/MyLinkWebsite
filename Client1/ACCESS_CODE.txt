                            HOW TO CHANGE ACCESS CODE
═══════════════════════════════════════════════════════════════════════════

1. Open config.js
2. Find line: magicWord: "Premium",
3. Change to your desired code, e.g.: magicWord: "MyCode123",
4. Save the file

The new access code will be active immediately!

═══════════════════════════════════════════════════════════════════════════
                            DEBUGGING TIPS
═══════════════════════════════════════════════════════════════════════════

If the access code still doesn't work:
1. Open browser DevTools (F12)
2. Go to Console tab
3. Try entering the access code
4. Check the console output for debug messages showing what was entered vs expected
5. Make sure there are no extra spaces or special characters
